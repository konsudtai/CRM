# FAQ — คำถามที่พบบ่อย

## ทั่วไป

### Q: บริษัทให้บริการอะไรบ้าง?
A: เราให้บริการด้าน Cloud Solutions, Software Development, IT Consulting, Managed Services และ Cybersecurity สำหรับธุรกิจ SMB ในประเทศไทย

### Q: มีสำนักงานอยู่ที่ไหน?
A: สำนักงานใหญ่อยู่ที่กรุงเทพฯ สามารถนัดหมายเข้าพบได้ในเวลาทำการ

### Q: รับงานต่างจังหวัดไหม?
A: รับครับ เราให้บริการทั่วประเทศ สามารถทำงาน Remote ได้ และมี On-site support สำหรับลูกค้า Enterprise

## ราคาและการชำระเงิน

### Q: ราคาเริ่มต้นเท่าไหร่?
A: ขึ้นอยู่กับบริการ:
- Cloud Migration: เริ่มต้น ฿150,000
- Managed Service: เริ่มต้น ฿25,000/เดือน
- Web App Development: เริ่มต้น ฿300,000
- IT Consulting: เริ่มต้น ฿80,000

### Q: มีส่วนลดไหม?
A: เรามีส่วนลดสำหรับสัญญารายปี และ package deals สำหรับลูกค้าที่ใช้หลายบริการ กรุณาติดต่อทีมขายเพื่อขอใบเสนอราคา

### Q: ชำระเงินอย่างไร?
A: รับชำระผ่านโอนธนาคาร, เช็ค, บัตรเครดิต เครดิตเทอม 30 วันสำหรับลูกค้าที่ผ่านการอนุมัติ

### Q: มี Free Trial ไหม?
A: สำหรับ Managed Cloud Service มี Free Assessment 1 ครั้ง และ Trial 14 วันสำหรับ Basic Plan

## เทคนิค

### Q: รองรับ Cloud อะไรบ้าง?
A: AWS (หลัก), Microsoft Azure, Google Cloud Platform

### Q: ใช้เทคโนโลยีอะไรในการพัฒนา?
A: Frontend: React, Next.js, Vue.js / Backend: Node.js, Python, Go / Database: PostgreSQL, MongoDB, Redis / Cloud: AWS (Lambda, ECS, RDS, S3)

### Q: มี SLA อย่างไร?
A: Managed Service: 99.9% Uptime SLA / Premium Support: Response ภายใน 4 ชม. / Enterprise: Response ภายใน 1 ชม.

## การสนับสนุน

### Q: ติดต่อ Support อย่างไร?
A: Email, โทรศัพท์, LINE OA ตามแพ็กเกจที่เลือก Basic = Email only, Premium = Phone + LINE 24/7

### Q: ถ้าระบบล่มทำอย่างไร?
A: สำหรับลูกค้า Managed Service — ระบบ Monitoring จะแจ้งเตือนอัตโนมัติ ทีมจะเข้าแก้ไขทันที ไม่ต้องแจ้งเอง

### Q: มีการ Training ให้ไหม?
A: มีครับ ทุก Project รวม Training ให้ทีมลูกค้า 1-2 วัน และมี Documentation ครบ
